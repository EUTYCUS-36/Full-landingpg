## üìù About This Project

**SMERTCO Furniture** is a stylish, clean, and professional landing page designed to showcase handcrafted, quality furniture with modern design and usability in mind.

Built over two weeks, this project reflects my continuous growth in frontend web development and showcases:

- Responsive, mobile-first design  
- Semantic and accessible HTML5 markup  
- Modern CSS3 features including animations and grid/flexbox layouts  
- JavaScript-powered interactivity for smooth user experience  
- ARIA roles and labels for enhanced accessibility  
- Cross-browser compatibility and optimized performance
